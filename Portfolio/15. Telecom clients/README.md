# Plan Recommendation

### Tools Used
scikit-learn, pandas, Matplotlib.

### Key Features
Classification, hyperparameter selection, and machine learning model selection.

### Task Description
The mobile operator Megaline discovered that many customers were using legacy plans. They wanted to build a system capable of analyzing customer behavior and offering them a new plan: "Smart" or "Ultra."

We have data on the behavior of customers who have already switched to these plans. We need to build a model for the classification problem that would select the appropriate plan.

### General Conclusion
We prepared the datasets. We trained the following models: decision tree, random forest, and logistic regression. We selected the hyperparameters. We generated the output. We also validated the model on a test set and verified its adequacy.
